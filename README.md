# S.O.G — Worldwide Travel & Documentation Services

A modern, responsive landing page for a travel and documentation business offering study abroad, visa applications, and worldwide travel services.

## Core Services

1. **Study Abroad & Admissions** — University applications, scholarships, student visas
2. **Bank Statements & Visa Applications** — Financial documentation and visa processing
3. **Tourist Visas** — China, Canada, USA, UK, Australia, and more
4. **Business Travel & International Jobs** — Work visas and global opportunities
5. **Worldwide Travel & Documentation** — End-to-end documentation support

## Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, Grid, Flexbox, responsive design
- **Vanilla JavaScript** — No frameworks, lightweight interactivity

## Features

- Fully responsive (mobile, tablet, desktop)
- Smooth scroll and scroll-triggered animations
- Mobile hamburger navigation
- Sticky header with scroll effect
- Search/filter bar for services
- Newsletter signup
- High-quality imagery (Unsplash)

## Getting Started

1. Open `index.html` in a browser, or
2. Serve locally: `npx serve .` or `python -m http.server 8000`

## File Structure

```
S.O.G/
├── index.html      # Main page
├── styles.css      # All styles
├── script.js       # Interactivity
└── README.md       # This file
```

## Customization

- **Colors**: Edit CSS variables in `:root` (e.g. `--color-primary`, `--color-accent`)
- **Fonts**: Swap Google Fonts in `<head>` and `--font-display`, `--font-body`
- **Content**: Update text and images directly in `index.html`
